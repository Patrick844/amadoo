## Key answer: Yes, the backend is the same for iOS and Android

Both apps talk to the same API server. The platform only affects the frontend UI.

## First things to do TODAY

1. Create Apple Developer account — https://developer.apple.com/programs/enroll/ ($99/year)
2. Create Supabase account — https://supabase.com (free)
3. Create Expo account — https://expo.dev (free)
4. Install Xcode from the Mac App Store (large download, start this now)
